La différence fondamentale entre une calculatrice et un ordinateur est qu'un ordinateur peut être programmé d'une manière qui permet au programme de prendre différentes branches en fonction des résultats intermédiaires, alors que les calculatrices sont conçues à l'avance avec des fonctions spécifiques intégrées. La distinction n'est pas nette: certains appareils classés comme calculatrices programmables disposent de fonctions de programmation, parfois avec un support pour les langages de programmation Find more on wikipedia page...

Dans cette série de missions, vous allez construire une calculatrice élémentaire.

En entrée, vous obtenez une séquence de touches pressées et, comme résultat de cette fonction, vous devez montrer ce qui sera affiché à l'écran lorsque la dernière touche sera pressée. Attention, ce n'est pas toujours le résultat de l'expression. En fait, jouer avec la calculatrice physique ou l'application vous aidera vraiment à repérer les cas limites.

Un espace maximal de 5 chiffres est ajouté. Cela signifie que le nombre maximum que vous pouvez entrer est "99999".

Comportement attendu:

  • les zéros du début doivent être supprimés, le nombre de zéros seulement est converti en zéro unique;
  • parmi les signes +- entre les nombres, le dernier doit être pris;
  • "==" signifie que la dernière opération doit être répétée;
  • "+=" ou "-=" - ajouter/soustraire le nombre (ou le résultat de l'opération) avant la combinaison (doubler le nombre/soustraire lui-même);
  • la calculatrice ignore les chiffres que vous entrez après le 5;
  • "-" pour les nombres < 0 ne prend PAS la place d'un chiffre;
  • si l'abs(résultat) est supérieur à 99999 - "erreur" est affiché comme résultat.
