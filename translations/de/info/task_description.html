Der grundlegende Unterschied zwischen einem Taschenrechner und einem Computer besteht darin, dass ein Computer so programmiert werden kann, dass das Programm je nach den Zwischenergebnissen verschiedene Verzweigungen annimmt, während Taschenrechner von vornherein mit bestimmten Funktionen ausgestattet sind. Die Unterscheidung ist nicht eindeutig: Einige Geräte, die als programmierbare Taschenrechner eingestuft werden, verfügen über Programmierfunktionen, manchmal mit Unterstützung für Programmiersprachen Find more on wikipedia page...

In dieser Aufgabenserie werden Sie einen einfachen Taschenrechner bauen.

Als Eingabe erhalten Sie eine Folge von gedrückten Tasten, und als Ergebnis dieser Funktion sollten Sie angeben, was auf dem Bildschirm angezeigt wird, wenn die letzte Taste gedrückt wird. Seien Sie aufmerksam, es ist nicht immer das Ergebnis des Ausdrucks. Wenn Sie mit dem physischen Taschenrechner oder der App spielen, können Sie die Randfälle besser erkennen.

Der maximale Ziffernabstand von 5 wird hinzugefügt. Das bedeutet, dass die maximale Zahl, die Sie eingeben können, "99999" ist.

Erwartetes Verhalten:

  • anfängliche Nullen sollten entfernt werden, Nur-Nullen-Zahlen - werden in eine einzelne Null umgewandelt;
  • von den +- Zeichen zwischen den Zahlen sollte das letzte genommen werden;
  • "==" bedeutet die Wiederholung der letzten Operation;
  • "+=" oder "-=" - addiert/subtrahiert die Zahl (oder das Ergebnis der Operation) vor der Kombination (verdoppelt die Zahl/subtrahiert sich);
  • der Rechner ignoriert die Ziffer, die Sie nach der 5 eingeben;
  • "-" für Zahlen < 0 nimmt NICHT die Stelle der Ziffer ein;
  • wenn abs(Ergebnis) größer als 99999 ist - wird "Fehler" als Ergebnis angezeigt.

Eingabe: Zeichenkette.

Ausgabe: String.

Beispiele: 

{{init_code_tmpl}}

Wie es verwendet wird: Taschenrechner sind weit verbreitet. Es ist interessant und nützlich, die Prinzipien ihrer Eingabe und Ausgabe zu verstehen.

Vorbedingung: Erlaubte Zeichen: Ziffern (0-9), Plus- (+), Minus- (-) oder Gleichheitszeichen (=) und deren Kombinationen (+=, +-, == usw.). Die Zahl darf nicht mehr als 5 Ziffern enthalten.
