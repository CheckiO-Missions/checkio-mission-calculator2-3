在本系列任务中，您将制作一个初级计算器。

作为输入，您需要按下一串按键，而作为该功能的结果，您应该显示按下最后一个按键时屏幕上将显示的内容。注意，这并不总是表达式的结果。实际上，使用实物计算器或应用程序确实有助于你捕捉边缘情况。

最大数字空间为 5。这意味着您可以输入的最大数字是 "99999"。

预期行为

  • 应删除开头的零，将仅有零的数字转换为单个零；
  • 数字之间的 +- 符号应取最后一个；
  • "=="表示重复上次操作；
  • "+="或"-="--在组合前对数字（或运算结果）进行加减运算（将数字加倍或减去本身）；
  • 计算器忽略在第 5 位之后输入的数字；
  • 数字 < 0 的"-"不占位数；
  • 如果 abs(result) 大于 99999，计算结果将显示 "错误"。
